# Digital Scrapbook

An interactive about me page built as a digital scrapbook instead of a traditional portfolio.

It is a small personal web experience where people can explore:
- where I live
- what I do
- the things I am obsessed with

The page includes draggable elements, photo folders, a looping intro video, and a short welcome screen to make the experience feel playful and personal.

## Built With
- HTML
- CSS
- JavaScript

## Concept
I wanted to create a little corner of the internet that feels more like me than a normal website or portfolio.

A tiny world shaped by creativity, tech, and the things I love.
